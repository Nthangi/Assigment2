# Quiz App - True/False History Edition

A simple Android quiz application that tests users' knowledge of historical facts through 10 true/false questions.

## Features

- **Quiz Flow**: Sequential navigation through 10 curated history questions
- **Immediate Feedback**: Visual feedback (green/red) for correct/incorrect answers
- **Score Tracking**: Final score display with percentage-based performance rating
- **Answer Review**: Detailed breakdown of incorrect answers post-quiz
- **Retry Capability**: Option to restart quiz without app restart
- **Responsive UI**: Clear question display and button states management

## Installation

1. Clone repository:
   ```bash
   git clone [repository-url]
   ```
2. Open project in Android Studio
3. Build project and run on emulator/device (API 21+ recommended)

## Usage

1. **Start Screen**: Tap "Start" to begin quiz
2. **Answer Questions**:
   - Read question carefully
   - Tap TRUE/FALSE button to answer
   - Receive immediate feedback
   - Press NEXT to continue
3. **Results Screen**:
   - View final score
   - Tap REVIEW to see incorrect answers
   - RETRY to restart quiz
   - EXIT to close app
